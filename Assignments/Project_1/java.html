Java

  • General purpose programming language.
    • This means it is designed to be applied to a wide variety of application domains.
  • Designed to have as few implementation dependencies as possible
  • Can run on all platforms that support Java without having to recompile thanks to the Java Virtual Machine
  • Development began at Sun Microsystems in 1991 by James Gosling
  • Latest release is Java 11
