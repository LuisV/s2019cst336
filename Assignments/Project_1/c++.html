C++

  • General purpose programming language.
    • This means it is designed to be applied to a wide variety of application domains.
  • Focuses on performance, efficiency and flexibility.
  • Heavily used in software infrastructure, e.g. JVM, and in applications where resources are scarce.
  • Development began at Bell Labs in 1979 by Bjarne Stroustrup
  • Latest release is C++17
“C++ is designed to allow you to express ideas, but if you don't have ideas or don't have any clue about how to express them, C++ doesn't offer much help.”
― Bjarne Stroustrup
